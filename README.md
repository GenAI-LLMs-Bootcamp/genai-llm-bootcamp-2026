This repository contains the lecture material, demo notebooks, and resources used in the Generative AI and LLM Bootcamp conducted at Amrita Vishwa Vidyapeetham, Bengaluru Campus.

## Bootcamp details

Organized by Amrita School of Computing, Amrita School of AI

Where :  Amrita Vishwa Vidyapeetham Bengaluru Campus

When :  1-4 April 2026

## What you will learn

- Foundations of AI and Generative Models
- Large Language Models and Applications
- Prompt Engineering and Retrieval Augmented Generation (RAG)
- Fine-Tuning and PEFT Techniques
- Multimodal AI including text, speech, and vision

## Prerequisites

To get started with the project files, you'll need to:
1. Setup [Google colab](https://colab.research.google.com/notebooks/intro.ipynb)
2. Or if running locally, install [Python](https://www.python.org) and [Jupyter Notebook](https://jupyter.org/)

   If you already have Python and pip installed, you can install Jupyter Notebook directly via the command line: 

    Open your terminal or Command Prompt.
    
    Run the installation command:
    
    ```bash
    pip install notebook
    ```
    
    Launch Jupyter Notebook: Navigate to the desired directory and run the command jupyter notebook in your terminal

4. [Hugging Face Tokens](https://huggingface.co/settings/tokens) 

## Quick Start for Participants

Participants can either run the notebooks locally or use [Google colab](https://colab.research.google.com/notebooks/intro.ipynb).

### Step 1 – Clone or download the repository
 ```bash
    git clone github.com/GenAI-LLMs-Bootcamp/genai-llm-bootcamp-2026.git
    cd genai-llm-bootcamp-2026
 ```
 Or alternatively, download the repository as a ZIP file.

### Step 2 – Install Python

Install **Python 3.10 or above**.

You can download Python from  
https://www.python.org

### Step 3 – Start Jupyter Notebook
Open the notebook corresponding to the workshop session.

Example- Day 01 Directory
